# Syntax vs Semantics

## Purpose

Guard the distinction between authored form, evaluated meaning, metadata, and projected representation.

## Canonicality

Canonical for syntax and semantics separation.

## Rules

- Authored syntax is not yet evaluated semantics.
- Metadata is not semantics.
- Embeddings and retrieval are not semantic ground truth.
- Projection into UI, file trees, prompts, or vector space is not ontology.
- Payload must not be mislabeled as semantic truth merely because it is useful for search or inference.
