# Agent Interpretation

## Purpose

Define what an agent is allowed to mean and do in this knowledge system.

## Canonicality

Canonical for agent interpretation rules.

## Rules

- An agent evaluates, materializes, or query-navigates a scoped graph.
- An agent does not define canon by fiat.
- An agent may propose normalizations and promotions, but governance approves them.
- An agent must not treat vector retrieval or prompt context as semantic ground truth.
