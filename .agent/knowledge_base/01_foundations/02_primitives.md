# Primitives

## Purpose

Define the minimal object vocabulary of the system.

## Canonicality

Canonical for primitive terms.

## Seed primitives

- object
- morphism
- wire
- scoped object
- identity
- graph state
- authored artifact
- payload
- metadata
- projection
- evaluated semantics
- version/log truth
- superposition

## Identity rule

The abstract actor or user is a primitive graph object. Concrete `userId` bindings are contingent instances and belong in `04_value_layer/identities.json`.
