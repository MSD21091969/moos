# Value Layer

## Purpose

Store contingent graph instances and environment-specific facts.

## Canonicality

Canonical for runtime and deployment facts, not for ontology.

## What belongs here

- providers
- benchmarks as concrete datasets or measurement instances
- preferences
- concrete identity bindings such as `userId`
- workstation and environment state
- platform and install distribution facts
- runtime surface bindings

## What does not belong here

- foundational ontology
- semantic definitions
- category language
- invariant rules
