# Tools Index

Tools are executable scripts or binary wrappers invoked by the GraphToolServer.
They are typically single-file Python scripts using `typer` or `argparse`.

## Recommended Tools

- **`db_migrate.py`**: Wrapper around Alembic for schema changes.
- **`generate_secrets.py`**: Utility to create JWT keys.
- **`rag_ingest.py`**: Script to index a folder of documents into the Vector DB.
