## FFS1 ColliderDataSystems Context

This plan operates within the FFS1 workspace structure:

```
FFS0_Factory/                          # Root factory
└── workspaces/
    └── FFS1_ColliderDataSystems/      # ← You are here
        ├── .agent/                     # IDE agent context
        │   ├── manifest.yaml           # Inheritance config
        │   ├── knowledge/
        │   │   ├── architecture/       # System docs
        │   │   └── devlog/             # Session logs (this file)
        │   └── instructions/           # Agent prompts
        │
        ├── FFS2_ColliderBackends.../   # Backend servers
        │   ├── ColliderDataServer/     # :8000 REST + SSE
        │   ├── ColliderGraphToolServer/# :8001 WebSocket + agents
        │   └── ColliderVectorDbServer/ # :8002 Vector search
        │
        └── FFS3_ColliderApplications...# Frontend apps
            └── collider-frontend/      # Nx monorepo
                ├── apps/portal/        # Next.js Portal app
                └── libs/               # Shared libraries
```

### .agent System

The `.agent` folder provides IDE context for Copilot/AI assistants:
- **manifest.yaml** - Declares inheritance from parent workspaces
- **instructions/** - Prompts for code generation
- **knowledge/** - Architecture docs and devlogs
- **configs/** - Domain and server configuration
- **exports** - Files inherited by child workspaces

Progress is logged in `.agent/knowledge/devlog/` to maintain context across sessions.
