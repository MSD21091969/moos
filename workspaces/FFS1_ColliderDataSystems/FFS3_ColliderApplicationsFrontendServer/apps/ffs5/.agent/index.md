# ffs5 Agent Context — Picture-in-Picture

Minimal floating overlay UI for quick agent interactions.

## Purpose

- Always-on-top compact agent chat
- Appears as a small floating window over other browser tabs
- Shares backend dependencies with ffs4/ffs6 (MOOS at `:8000`, `:8004`, `:18789`)

Inherits from FFS3 workspace `.agent`.
