# Codebase: FFS3 ColliderFrontend

> Main React Application Server (Vite).

## Role

Hosts the web-based "AppNodes" when they are not running inside the Chrome Extension sidepanel or are being developed in isolation.

## Structure

Contains sub-directories for specific App Contexts (FFS4-10), acting as a monorepo for frontend applications.

## Key Apps

- **FFS4 Sidepanel**: The browser companion UI.
- **FFS6 IDE**: The web-based visualization of the file system.
- **FFS7 Admin**: The account management dashboard.
